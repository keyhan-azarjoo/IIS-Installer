# Proxy

This folder is self-contained. The active layout is:

- `common/`: shared management and uninstall scripts
- `layers/`: install and per-layer management scripts
- `panel/`: local proxy management panel sources
- `docs/`: imported setup and reference documents
- `windows/`: Windows entrypoint
- `linux-macos/`: Linux/WSL entrypoint

`source/` is obsolete and should not be used by the installers.
